Mesh Network

Mesh Network ဆိုတာဘာလည်း

Mesh Network ဆိုတာ ဆက်သွယ်ရေး ပစ္စည်းတွေ တစ်ခုနဲ့တစ်ခု အကွာအဝေးတစ်ခု အတွင်း တိုက်ရိုက်ချိတ်ဆက်ပြီး ကွန်ယက်ဖြစ်အောင် လုပ်တာပါ။ အဓိကအားဖြင့် Bluetooth ကို အခြေခံပြီး ချိတ်ဆက်တာမို့ အင်တာနက် ပြတ်တောက်နေချိန်တွေမှာ အသုံးဝင်ပါတယ်။အင်တာနက်မရှိတဲ့‌နေရာတွေမှာ ကျွန်မတို့ကိုယ်တိုင် Mesh Network တစ်ခုအဖြစ် ဆက်သွယ်ရေးလိုင်းတွေ ဖြစ်အောင် လုပ်လို့ရပါတယ်။ ဒါပေမယ့် လူညီမှ ရမှာပါ။ Mesh Network သဘောတရားကိုက ပုံမှာပြထားသလိုဘဲ အနီးအနားက လူ‌တွေကနေ တစ်ဆင့်ပြီးတစ်ဆင့် နီးစပ်ရာကို ဖြန့်ကျက်သွားတာပါ။ ဒါကြောင့် လူတိုင်း နေရာတိုင်း Mesh Network App တစ်ခုသုံးဖို့ လိုပါတယ်။ ဒါဆိုရင် အင်တာနက်မရှိလည်း ဆက်သွယ်မှု သတင်းဖြန့်‌ဝေမှုတွေ မပြတ်တောက်တော့ပါဘူး။

Mesh Network ဖြစ်ဖို့ ဘာတွေလိုလဲ

လူတိုင်း Bridgefy/ Briar လိုမျိုး တူညီတဲ့ Mesh Network App လိုမျိုး သုံးပါ။ Bluetooth သို့မဟုတ် WiFi တွေ ဖွင့်ပါ။ Mesh Network App တွေက အကွာအဝေး သတ်မှတ်ချက် (Briar ဆို ၂၀ မီတာ) တွေရှိတာကြောင့် လူများများ သုံးပါ။ လူများလေ တစ်ဆင့်ပြီး တစ်ဆင့် ပိုချိတ်ဆက်နိုင်လေပါ။ Mesh Network ‌App တွေကနေ အင်တာနက်လိုင်းဖွင့်ပြီး (အနည်းဆုံး တစ်ခါ) ပြောထားပါ။ App ကို နောက်ဆုံး version သုံးပါ။

VPN

VPN ဆိုတာဘာလည်း

Virtual Private Network လို့ခေါ်တဲ့ VPN ဆိုတာ အင်တာနက်လုံခြုံရေး၀န်ဆောင်မှုတစ်ခုပဲ ဖြစ်ပါတယ်။ VPN software (သို့) Server တစ်ခုရဲ့ အလုပ်လုပ်ပုံကို ရှင်းပြရမယ်ဆိုရင်တော့ VPN ဟာ သင့်ရဲ့ အွန်လိုင်းဆက်သွယ်မှု လုပ်ဆောင်ချက်တွေကို အကာအကွယ်ပေးခြင်းကြောင့် သင်အသုံးပြုနေတဲ့ အင်တာနက်ကွန်ယက်ထဲသို့ ပြင်ပမှ ကြားဖြတ် ၀င်ရောက် ကြည့်ရှုနိုင်မှာမဟုတ်ပါ။ ဒါ့အပြင် VPN များဟာ သင်ရဲ့အင်တာနက်အသုံးပြုမှု လမ်းကြောင်းကို လုံခြုံစေပြီး မသမာသူများ၊ အင်တာနက် ၀န်ဆောင်မှုပေးသူများ (Internet service providers) နှင့် ကြော်ငြာ Website များထံမှ သင့်ရဲ့ ကြည့်ရှုခြင်းအချက်အလက်တွေကို ဖုံးကွယ်ပေးထားပါတယ်။ း

VPN ကိုဘာကြောင့်သုံးသင့်တာလည်း

လက်တွေ့မှာ အများပြည်သူသုံး Public Wi-Fi တွေဟာ လုံခြုံမှုမရှိနိုင်ပါဘူး။ ဒါကြောင့် Public Wifi တွေနဲ့ ချိတ်ဆက်လိုက်မိတဲ့ အခါ ကျွန်မတို့ရဲ့ ဖုန်းတွေ Laptop တွေက လုံခြုံမှုအတွက် ကာကွယ်ထားတဲ့အရာ ဘာမှမရှိသလို ဖြစ်နေမှာပါ။ Mobile Banking Apps များတွင် password ရိုက်နှိပ်ခြင်း၊ ကိုယ်ရေးအချက်အလက်ဖြည့်သွင်းခြင်း၊ Facebook အကောင့်အား Password ရိုက်၍ ၀င်ရောက်ခြင်း အစရှိတဲ့ အရာတွေကို Public Wifi ချိတ်ပီး သုံးစွဲမိပါက မသမာသူ hacker တွေက အလွယ်တကူ ထိုးဖောက်ပီး အရေးကြီးကိုယ်ရေးအချက် အလက် တွေကို ခိုးယူသွားနိုင်ပါတယ်။ VPN ကို အသုံးပြုသင့်တဲ့ အကြောင်းရင်းကတော့ အပေါ်က အန္တရာယ်တွေလိုမျိုးတွေကို ကြိုတင်ကာကွယ်ဖို့ပဲ ဖြစ်ပါတယ်။ VPN ကိုအသုံးပြုပြီး အင်တာနက်ကို ပိုမိုလုံခြုံစွာ အသုံးပြုနိုင်တဲ့အပြင် တားမြစ်ခံရတဲ့ Website တွေကို ကြည့်ရှုနိုင်ပါတယ်။ VPN က ကျွန်မတို့ရဲ့ တကယ့် IP address ကို ဝှက်ထားပေးပြီး ပိတ်ထားတဲ့ လုံခြုံမှုမရှိတဲ့ website တွေကို ကြည့်လို့ရအောင်၊ လုံခြုံမှုရှိအောင် ကူညီပေးပါတယ်။ VPN သုံးထားမယ်ဆိုရင် ပုံမှန်သုံးနေကြ မိုဘိုင်း အေပာ်ရိတ်တာ (သို့) အင်တာနက်၀န်ဆောင်မှုတွေထက် ပိုလုံခြုံတဲ့ Private Server တစ်ခုနဲ့ ချိတ်ဆက်ပေးပါတယ်။ IP Address ကို ကာကွယ်ပေးရုံသာမက သင်ဘယ်သူလဲဆိုတာ မသိရအောင် လုံခြုံမှု အတိုင်းအတာ တစ်ခုထိ ပေးစွမ်းနိုင်ပါတယ်။

VPN သုံးရင် သတိထားရမယ့် အချက်များ

Free VPN Apps တွေကို အသုံးပြုတာထက် Paid VPN apps တွေကို အသုံးပြုတာက ကျွန်မတို့ရဲ့ သတင်းအချက်တွေအတွက် ပိုမို လုံခြုံစိတ်ချရတာကို သတိပြုပါ။ တချို့သော VPN apps များတွင် VPN ကျော်၍ အသုံးပြုချိန်မှာ မူလ အင်တာနက်နှုန်းထက် bandwidth လျော့သွားမှုကြောင့် အင်တာနက်နှေးသွားတာမျိုး ကြုံရတတ်ပါတယ်။ VPN အသုံးပြုတဲ့အခါ နိုင်ငံတစ်ခုကို ရွေးချယ် connect လုပ်ရတာဖြစ်ပီး မိမိရဲ့ Account တစ်ခုတည်းနဲ့ Mobile Phone၊ Laptop နှင့် Tablet အစရှိတဲ့ Digital Device တွေကို တပြိုင်တည်း ချိတ်ဆက် သုံးစွဲနိုင်တာကြောင့် Device အားလုံးမှာ နိုင်ငံတစ်မျိုးတည်းသာ ရွေးချယ်ဖို့လိုပါတယ်။ Facebook အသုံးပြုနေစဉ်မှာ device နှစ်ခုမှာ location နှစ်ခု မတူတာရွေးထားတာမျိုးဖြစ်ဖြစ်၊ VPN အသုံးပြုနေစဉ်မှာ Facebook အကောင့်ကို log in, log out ပြုလုပ်တာမျိုး ဖြစ်ဖြစ် လုပ်မိပါက ယာယီ Lock ကျတာမျိုး ဖြစ်တတ်ပါတယ်။
